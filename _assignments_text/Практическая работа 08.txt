Практическая работа 08 Полносвязная нейронная сеть для классификации изображений Полносвязная нейронная сеть (fully connected neural network, FCNN или MLP - multilayer perceptron) может быть применена для классификации изображений, хотя и с определенными ограничениями. Процесс классификации включает следующие этапы: 1. Векторизация изображения: Двумерная матрица пикселей, представляющая изображение, преобразуется в одномерный вектор. Для grayscale изображения размером m x n результирующий вектор будет иметь размерность m * n. Для цветного изображения (например, RGB) с тремя цветовыми каналами размерность вектора будет m * n * 3. Эта процедура, называемая flattening, приводит к потере пространственной информации о взаимном расположении пикселей, что является одним из основных недостатков применения FCNN для анализа изображений. 2. Архитектура и функционирование сети: FCNN состоит из входного, одного или нескольких скрытых и выходного слоев. Входной слой: Размерность входного слоя определяется размерностью вектора изображения. Каждый нейрон входного слоя соответствует одному элементу вектора. Скрытые слои: Каждый нейрон скрытого слоя связан со всеми нейронами предыдущего слоя. Выход каждого нейрона j в слое l вычисляется как сумма произведений каждого входного параетра с соответствующи весом и преобразованная с помощью функции активакции. Выходной слой: Количество нейронов выходного слоя K соответствует количеству классов. Для многоклассовой классификации обычно используется функция активации softmax, которая нормализует выходные значения в вероятностное распределение: 3. Процесс обучения: Обучение FCNN заключае тся в оптимизации весов w для минимизации функции потерь, которая измеряет разницу между предсказанными значениями ŷ и истинными метками y. Оптимизация обычно выполняется с помощью методов градиентного спуска (например, stochastic gradient descent, Adam). 4. Классификация: После обучения сеть может классифицировать новые изображения. Векторизованное изображение подается на вход сети, и на выходе получается вектор предсказанных вероятностей для каждого класса. Изображение классифицируется как принадлежащее к классу с наибольшей вероятностью. Недостатки FCNN для классификации изображений: Игнорирование пространственной структуры: Векторизация разрушает пространственные связи между пикселями, что критично для анализа изображений. Высокая размерность входного вектора: Большие изображения приводят к векторам высокой размерности, что увеличивает количество параметров сети и вычислительную сложность, повышая риск переобучения. T ensorFlow и Keras TensorFlow — это мощная библиотека с открытым исходным кодом для численного вычисления и крупномасштабного машинного обучения. Keras — это высокоуровневый API нейронных сетей, написанный на Python, который работает поверх TensorFlow (и других бэкендов). Keras упрощает разработку и обучение нейронных сетей, предоставляя интуитивно понятный и модульный интерфейс. Начиная с TensorFlow 2.0, Keras стал предпочтительным способом построения моделей в TensorFlow. import tensorflow as tf from tensorflow import keras from tensorflow.keras.layers import Dense, Flatten import matplotlib.pyplot as plt # 1. Загрузка и предобработка данных MNIST (x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data() # Нормализация пикселей (0-1) x_train = x_train.astype("float32") / 255.0 x_test = x_test.astype("float32") / 255.0 # Создание массива меток классов y_train = keras.utils.to_categorical(y_train, num_classes=10) y_test = keras.utils.to_categorical(y_test, num_classes=10) # 2. Построение модели model = keras.Sequential([ Flatten(input_shape=(28, 28)), # Преобразование 2D изображения в 1D вектор Dense(128, activation='relu'), # Полносвязный слой с 128 нейронами и ReLU активацией Dense(10, activation='softmax') # Выходной слой с 10 нейронами (для 10 цифр) и softmax активацией ]) # 3. Компиляция модели model.compile( optimi zer='adam', # Оптимизатор Adam loss='categorical_crossentropy', # Функция потерь для многоклассовой классификации metrics=['accuracy'] # Метрика для оценки качества - точность ) # 4. Обучение модели history = model.fit( x_train, y_train, epochs=10, # Количество эпох обучения batch_size=32, # Размер батча validation_split=0.1 # 10% данных для валидации (контроль переобучения) ) # 5. Оценка модели loss, accuracy = model.evaluate(x_test, y_test) print(f"Loss: {loss}, Accuracy: {accuracy}") # Вывод результатов обучения plt.figure(figsize=(12, 4)) plt.subplot(1, 2, 1) plt.plot(history.history['accuracy'], label='Training Accuracy') plt.plot(history.history['val_accuracy'], label='Validation Accuracy') plt.xlabel('Epoch') plt.ylabel('Accuracy') plt.legend() plt.title('Training and Validation Accuracy') plt.subplot(1, 2, 2) plt.plot(history.history['loss'], label='Training Loss') plt.plot(history.history['val_loss'], label='Validation Loss') plt.xlabel('Epoch') plt.ylabel('Loss') plt.legend() plt.title('Training and Validation Loss') plt.show() keras.datasets.mnist.load_data(): Загружает набор данных MNIST. Возвращает кортежи обучающих и тестовых данных (изображения и метки). astype (" float 32") / 255.0: Нормализует значения пикселей к диапазону [0, 1]. keras . utils . to _ categorical ( y _ train , num _ classes =10): Создание мвссива меток . num _ classes - количество классов. keras . Sequential (): Создает последовательную модель, в которую слои добавляются один за другим. PyTorch PyTorch — это библиотека глубокого обучения с открытым исходным кодом, разработанная Facebook. Она предоставляет два основных уровня функциональности: Тензоры: Многомерные массивы, похожие на NumPy arrays, но с возможностью вычислений на GPU для ускорения обучения. Автоматическое дифференцирование: Автоматически вычисляет градиенты для оптимизации параметров модели во время обучения. PyTorch популярен благодаря своей гибкости, динамическому построению графа вычислений и активному сообществу. import torch import torch.nn as nn import torch.optim as optim from torchvision import datasets, transforms # 1. Загрузка и предобработка данных MNIST transform = transforms.Compose([transforms.ToTensor(), transforms.Normalize((0.1307,), (0.3081,))]) train_dataset = datasets.MNIST(root='./data', train=True, download=True, transform=transform) test_dataset = datasets.MNIST(root='./data', train=False, transform=transform) train_loader = torch.utils.data.DataLoader(train_dataset, batch_size=64, shuffle=True) test_loader = torch.utils.data.DataLoader(test_dataset, batch_size=1000, shuffle=False) # 2. Определение модели class Net(nn.Module): def __init__(self): super(Net, self).__init__() self.fc1 = nn.Linear(784, 128) # Полносвязный слой 784 -> 128 self.relu = nn.ReLU() # Функция активации ReLU self.fc2 = nn.Linear(128, 10) # Полносвязный слой 128 -> 10 def forward(self, x): x = x.view(-1, 784) # Преобразование изображения в вектор x = self.fc1(x) x = self.relu(x) x = self.fc2(x) return x # Logits - сырые выходные данные, softmax применяется позже # 3. Создание экземпляра модели, оптимизатора и функции потерь model = Net() optimizer = optim.Adam(model.parameters(), lr=0.001) # Adam optimizer, learning rate = 0.001 criterion = nn.CrossEntropyLoss() # Cross-entropy loss (включает softmax) # 4. Обучение модели num_epochs = 10 for epoch in range(num_epochs): for i, (data, target) in enumerate(train_loader): optimizer.zero_grad() # Обнуление градиентов output = model(data) # Прямой проход loss = criterion(output, target) # Расчет потерь loss.backward() # Обратный проход (вычисление градиентов) optimizer.step() # Обновление весов # 5. Тестирование модели correct = 0 total = 0 with torch.no_grad(): # Без вычисления градиентов for data, target in test_loader: output = model(data) _, predicted = torch.max(output.data, 1) # Получение предсказанного класса total += target.size(0) correct += (predicted == target).sum().item() print(f'Accuracy: {100 * correct / total}%') Описание функций и параметров: transforms.Compose(): Комбинирует несколько преобразований. transforms.ToTensor(): Преобразует PIL Image или numpy.ndarray в тензор. transforms.Normalize((mean,), (std,)): Нормализует тензор с заданным средним и стандартным отклонением. datasets.MNIST(root, train, download, transform): Загружает набор данных MNIST. root - путь к папке с данными. train - True для обучающего набора, False для тестового. download - True для загрузки данных, если их нет. transform - преобразования, применяемые к данным. torch.utils.data.DataLoader(dataset, batch_size, shuffle): Создает загрузчик данных. dataset - набор данных. batch_size - размер батча. shuffle - True для перемешивания данных. nn.Linear(in_features, out_features): Полносвязный слой. in_features - количество входных признаков. out_features - количество выходных признаков. nn.ReLU(): Функция активации ReLU. nn.CrossEntropyLoss(): Функция потерь кросс-энтропии (включает softmax). optim.Adam(params, lr): Оптимизатор Adam. params - параметры модели для оптимизации. lr - learning rate. optimizer.zero_grad(): Обнуляет градиенты перед каждым шагом оптимизации. loss.backward(): Вычисляет градиенты. optimizer.step(): Обновляет веса модели. torch.no_grad(): Отключает вычисление градиентов во время тестирования. torch.max(input, dim): Возвращает максимальное значение и его индекс вдоль заданной размерности dim.
